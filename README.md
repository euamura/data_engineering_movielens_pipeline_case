# Netflix Data Pipeline 🎬

**[EN]** End-to-end data engineering pipeline built on MovieLens rating and recommendation data — covering raw ingestion on Google Cloud Storage, dimensional modeling in BigQuery (Bronze → Silver → Gold), containerized deployment with Docker, and business intelligence dashboards in Metabase. The project goes beyond infrastructure: it includes a model evaluation layer that measured recommendation quality metrics (MAE, RMSE, Bias) and, given the poor initial results, produced a business-facing analysis proposing a new model architecture. → [Full README (English)](docs/README_EN.md)

**[PT]** Pipeline de engenharia de dados ponta a ponta construído sobre dados de avaliações e recomendações da MovieLens — cobrindo ingestão bruta no Google Cloud Storage, modelagem dimensional no BigQuery (Bronze → Silver → Gold), deploy containerizado com Docker e dashboards de business intelligence no Metabase. O projeto vai além da infraestrutura: inclui uma camada de avaliação do modelo de recomendação com métricas de qualidade (MAE, RMSE, Bias) e, diante dos resultados insatisfatórios, produziu uma análise orientada a negócio propondo uma nova arquitetura de modelo. → [README Completo (Português)](docs/README_PT.md)
